Интеграции

Данные — туда, где вы их смотрите.

Кабинет не заменяет BI-систему. Отгружаем сырые логи и агрегированные метрики в ваше хранилище или инструмент визуализации.

API

REST API с токеном. Выгрузка показов, кликов и конверсий — по любому срезу, в любой период.

BigQuery / S3

Ежедневная выгрузка сырых логов в ваше хранилище. Подключаем Looker, Metabase, Power BI.

GA4 / Яндекс.Метрика

Post-click атрибуция по UTM. Сопоставляем показы с визитами и покупками в вашей веб-аналитике.

MMP

AppsFlyer, Adjust, Airbridge, myTracker. Полная мобильная атрибуция с корректной работой через SKAdNetwork.
